Entendendo Comunicação Client x Server

  • História Web

    • A Web surgiu no final dos anos 80 e começo dos anos 90 por causa de um cara chamado Tim Berners-Lee, que trabalhava no CERN,
    Foi baseada na ArpaNet criada durante a guerra fria. O primeiro navegador criado foi chamado de WorldWideWeb

  • O que são Clients

    • Clients são dispositivos ou softwares que fazem requisições a um servidor para obter algum serviço ou informação. Eles são a "parte da frente" da interação entre Client x Servers

  • O que são Servers

    • Servers são computadores com programas que fornecem serviços, dados ou recursos para clients. Eles ficam rodando constantemente, esperando por requisições e respondendo a elas.

  • Linguagem de Programação

    • Para o Client:

    • JavaScript: A principal linguagem para interatividade e dinamismo.
    • HTML: Linguagem de marcação para estruturar páginas.
    • CSS: Usada para estilizar páginas.

    Para o Server:

    • Node.js: Plataforma para rodar JavaScript no servidor.
    • PHP: Linguagem de programação para servidores.
    • C#: Linguagem de programação para servidores.
      • "Dentre outras linguagens"

Introduçao Html

  • Estrutura Básicas Html

    • HTML é uma linguagem de marcação que é usada para estruturar páginas web. Ela é composta por tags que são interpretadas pelo navegador para exibir o conteúdo da página, por exemplo, tags básicas sao "head", "html", "body" .

  • Tags Html

    • As tags são elementos que compõem o HTML. Elas são formadas por um nome e podem ter atributos. As tags são usadas para estruturar o conteúdo da página.

  • Textos

    • Para adicionar texto a uma página HTML, você pode usar a tag "p" para parágrafos, a tag "h1" para títulos, a tag "strong" para texto em negrito e a tag "i" para texto em itálico.

  • Listas Ordenadas e Não Ordenadas

    • Para criar listas em HTML, você pode usar as tags "ul" para listas não ordenadas e "ol" para listas ordenadas. Dentro dessas tags, você pode usar a tag "li" para cada item da lista.
